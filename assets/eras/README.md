# Era image assets

Put one image for each Era in this folder. The site will automatically use these files in:

- `era-gallery.html`
- the quiz result page
- the generated share card

Expected filenames:

- `taylor-swift.jpg`
- `fearless.jpg`
- `speak-now.jpg`
- `red.jpg`
- `1989.jpg`
- `reputation.jpg`
- `lover.jpg`
- `folklore.jpg`
- `evermore.jpg`
- `midnights.jpg`
- `ttpd.jpg`
- `showgirl.jpg`

Use images you have the right to publish. If a file is missing, the page falls back to the original generated visual style.
